定位

2019-11-02 2 min read # CSS

定义的三种机制

  • 普通流 static： 元素的默认属性，没有定位，遵循文档流。

  • 绝对定位 absolute：被绝对定位的元素，会彻底脱离文档流，其原本在流中占据的位置被移除。

  • 相对定位：相对定位的元素以原本所在的位置进行偏移，其原本在流中的位置被保留。

绝对定位

absolute

* 如果父元素也是使用定位的元素，则在定位时，以父元素作为参照物，否则以根元素作为参照物。
* 当以根元素作为参照物时，top 定位时，则以左上角为起始位置进行定位。
* 当以根元素作为参照物时，bottom 定位时，则是以**窗口**的左下角进行定位.

单位

  • 百分比： 当偏移的值以 % 作为单位时，其实际移动的偏移量为 父元素宽｜高 * 百分比。

  • 例如：

    <style>
        #app {
            height: 500px;
            width: 500px;
            background-color: blue;
        }

        #demo {
            height: 100px;
            width: 100px;
            background-color: green;
            position: relative;
            left: 10%; // 实际向右偏移50px。
        }
    <style>

    <div id="app">
        <div id="demo"></div>
    </div>
